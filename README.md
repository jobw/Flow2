# Flow2


Flow 2 is een project dat met een groepje is uitgevoerd in opdracht van het bedrijf Sodaq. 
Het project betrof het ontwerpen van een CO2 meter met behulp van een ESP32.

Het project bestaat de Arduino code die op de ESP32 met een SCD30 sensor gebruikt word, en een website die de gegevens laat zien.
Het is een prototype gemaakt door redelijk onevervaren op dit gebied, veel zal dus niet optimaal zijn ;)
